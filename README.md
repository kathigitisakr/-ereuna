## 🚀 Οδηγίες για GitHub Pages

1. Δημιουργήστε νέο repository στο GitHub
2. Ανεβάστε όλα τα αρχεία αυτού του φακέλου
3. Πηγαίνετε σε **Settings → Pages**
4. Επιλέξτε **Source: Deploy from a branch**
5. Επιλέξτε **Branch: main** και **/ (root)**
6. Η σελίδα σας θα είναι διαθέσιμη στο `https://ΟΝΟΜΑ-ΣΑΣ.github.io/ΟΝΟΜΑ-REPO/`
