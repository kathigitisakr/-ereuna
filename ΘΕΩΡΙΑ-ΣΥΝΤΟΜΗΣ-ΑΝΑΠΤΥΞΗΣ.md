## 1. Μεταβλητές και Χαρακτηριστικά
*Απαραίτητο για: Ερώτηση 1*

### Τι είναι Μεταβλητή;
**Μεταβλητή** = Ένα χαρακτηριστικό που μπορεί να πάρει διαφορετικές τιμές.

### Τι είναι Χαρακτηριστικά (Attributes);
**Χαρακτηριστικά** = Οι συγκεκριμένες τιμές/κατηγορίες που παίρνει μια μεταβλητή.

### Κανόνες για Χαρακτηριστικά:
1. **Αμοιβαία αποκλειστικότητα**: Κάθε περίπτωση ανήκει σε ΜΙΑ μόνο κατηγορία
2. **Εξαντλητικότητα**: Οι κατηγορίες καλύπτουν ΟΛΕΣ τις πιθανές περιπτώσεις

### Παραδείγματα Κοινωνιολογικών Μεταβλητών:

| Μεταβλητή | Χαρακτηριστικά |
|-----------|----------------|
| Φύλο | Άνδρας, Γυναίκα |
| Οικογενειακή κατάσταση | Έγγαμος, Άγαμος, Διαζευγμένος, Χήρος |
| Εκπαίδευση | Δημοτικό, Γυμνάσιο, Λύκειο, Πανεπιστήμιο |
| Επάγγελμα | Εκπαιδευτικός, Γιατρός, Μηχανικός... |
| Θρησκεία | Χριστιανός, Μουσουλμάνος, Εβραίος... |
| Πολιτική προτίμηση | ΝΔ, ΣΥΡΙΖΑ, ΠΑΣΟΚ, Άλλο |
| Εισόδημα | Χαμηλό, Μεσαίο, Υψηλό |
| Ηλικία | 18-25, 26-35, 36-45... |

## 2. Σκοποί Έρευνας
*Απαραίτητο για: Ερώτηση 2*

### Οι Τρεις Βασικοί Σκοποί:

#### 2.1 Διερευνητική Έρευνα (Exploratory)
- **Ερώτημα**: Τι συμβαίνει; Τι υπάρχει;
- **Πότε**: Όταν το θέμα είναι νέο ή ελάχιστα μελετημένο
- **Στόχος**: Εξοικείωση με το φαινόμενο
- **Παράδειγμα**: "Ποιες είναι οι εμπειρίες των εκπαιδευτικών με την τηλεκπαίδευση κατά την πανδημία;"

#### 2.2 Περιγραφική Έρευνα (Descriptive)
- **Ερώτημα**: Τι; Πότε; Πού; Πώς; Πόσο;
- **Στόχος**: Περιγραφή χαρακτηριστικών/κατάστασης
- **Παράδειγμα**: "Πόσοι μαθητές έχουν πρόσβαση στο διαδίκτυο; Ποιο είναι το εκπαιδευτικό τους υπόβαθρο;"

#### 2.3 Ερμηνευτική/Αιτιολογική Έρευνα (Explanatory)
- **Ερώτημα**: **ΓΙΑΤΙ;**
- **Στόχος**: Εύρεση αιτιακών σχέσεων
- **Παράδειγμα**: "Γιατί κάποιοι μαθητές αποτυγχάνουν στα μαθηματικά;"

### Πίνακας Σύγκρισης:

| Σκοπός | Ερώτημα | Αποτέλεσμα |
|--------|---------|------------|
| Διερευνητικός | Τι; | Υποθέσεις, Ιδέες |
| Περιγραφικός | Πόσο; Πού; Πότε; | Στατιστικά, Περιγραφές |
| Ερμηνευτικός | Γιατί; | Αιτιακές σχέσεις |

## 3. Εννοιολόγηση
*Απαραίτητο για: Ερώτηση 3*

### Τι είναι η Εννοιολόγηση;
Η διαδικασία **ορισμού** μιας αφηρημένης έννοιας ώστε να γίνει μετρήσιμη.

### Δομή Εννοιολόγησης:

```
ΑΦΗΡΗΜΕΝΗ ΕΝΝΟΙΑ
       │
       ├── Ορισμός
       │
       └── ΔΙΑΣΤΑΣΕΙΣ
             │
             ├── Διάσταση 1 → Δείκτες → Ερωτήσεις
             ├── Διάσταση 2 → Δείκτες → Ερωτήσεις
             └── Διάσταση 3 → Δείκτες → Ερωτήσεις
```

### Βήματα Εννοιολόγησης:
1. **Ορισμός** της έννοιας
2. **Αναγνώριση διαστάσεων** (υπο-έννοιες)
3. **Επιλογή δεικτών** για κάθε διάσταση
4. **Δημιουργία ερωτήσεων** για κάθε δείκτη

### Παράδειγμα - Κακοποίηση Παιδιών:

| Στοιχείο | Περιεχόμενο |
|----------|-------------|
| **Ορισμός** | Κακομεταχείριση ανηλίκων από γονείς/κηδεμόνες |
| **Διάσταση 1** | Σωματική κακοποίηση |
| **Διάσταση 2** | Συναισθηματική κακοποίηση |
| **Διάσταση 3** | Ψυχολογική κακοποίηση |
| **Διάσταση 4** | Παραμέληση |

**Δείκτες (παραδείγματα ερωτήσεων):**
- Σωματική: "Σας έχει χτυπήσει γονέας σας;"
- Συναισθηματική: "Σας έχουν αποκαλέσει ηλίθιο;"
- Ψυχολογική: "Σας έχουν απειλήσει;"
- Παραμέληση: "Σας άφηναν μόνο για πολλές ώρες;"

## 4. Κλίμακες/Επίπεδα Μέτρησης
*Απαραίτητο για: Ερώτηση 4*

### Τα 4 Επίπεδα Μέτρησης:

#### 4.1 Ονομαστικό (Nominal)
- **Χαρακτηριστικό**: Κατηγορίες ΧΩΡΙΣ σειρά
- **Πράξεις**: Μόνο καταμέτρηση, ισότητα (=, ≠)
- **Παράδειγμα**: Φύλο (Άνδρας/Γυναίκα), Ναι/Όχι

#### 4.2 Διατακτικό (Ordinal)
- **Χαρακτηριστικό**: Κατηγορίες ΜΕ σειρά
- **Πράξεις**: Σύγκριση (>, <)
- **Παράδειγμα**: Μικρό < Μεσαίο < Μεγάλο

#### 4.3 Διαστημάτων (Interval)
- **Χαρακτηριστικό**: Ίσα διαστήματα, αυθαίρετο 0
- **Πράξεις**: +, -, μέσος όρος
- **Παράδειγμα**: Θερμοκρασία °C (0°C δεν σημαίνει "καθόλου θερμότητα")

#### 4.4 Αναλογικό (Ratio)
- **Χαρακτηριστικό**: Ίσα διαστήματα, ΑΠΟΛΥΤΟ 0
- **Πράξεις**: ×, ÷, αναλογίες ("διπλάσιο")
- **Παράδειγμα**: Ηλικία, αριθμός παιδιών (0 = κανένα)

### Παράδειγμα - Μέγεθος Οικογένειας:

| Επίπεδο | Ερώτηση/Μέτρηση |
|---------|-----------------|
| **Ονομαστικό** | "Έχετε αδέρφια;" (Ναι/Όχι) |
| **Διατακτικό** | "Πόσα άτομα;" (Λίγα/Μέτρια/Πολλά) ή (0-2 / 3-5 / 6+) |
| **Αναλογικό** | "Πόσα άτομα ακριβώς;" (0, 1, 2, 3, 4...) |

## 5. Διαχείριση Ελλιπών Δεδομένων
*Απαραίτητο για: Ερώτηση 5*

### Στρατηγικές Αντιμετώπισης:

| Στρατηγική | Περιγραφή | Πότε |
|------------|-----------|------|
| **Εξαίρεση (Listwise)** | Αφαίρεση περιπτώσεων | Λίγα ελλιπή, τυχαία |
| **Απόδοση μέσης τιμής** | Αντικατάσταση με Μ.Ο. | Λίγα, MCAR |
| **Ανάλυση ελλιπών** | Διερεύνηση μοτίβων | Όταν έχουν νόημα |
| **Πολλαπλή αντικατάσταση** | Στατιστικές μέθοδοι | Πολλά ελλιπή |
| **Χρήση προσέγγισης** | Εκτίμηση από άλλες μεταβλητές | Υπάρχει συσχέτιση |

### Ερωτήσεις για Απόφαση:
1. Πόσα είναι τα ελλιπή; (% του δείγματος)
2. Είναι τυχαία ή συστηματικά;
3. Ποια ανάλυση θα κάνουμε;
4. Τι σημαίνει η απουσία απάντησης;

## 6. Δειγματοληψία και Πιθανότητες
*Απαραίτητο για: Ερωτήσεις 6, 7*

### Τύποι Δειγματοληψίας:

#### Πιθανοτική:
- Κάθε μέλος έχει **γνωστή** πιθανότητα επιλογής
- Επιτρέπει **στατιστική γενίκευση**
- Τύποι: Απλή τυχαία, Συστηματική, Στρωματοποιημένη, Κατά συστάδες

#### Μη Πιθανοτική:
- **Άγνωστη** πιθανότητα επιλογής
- ΔΕΝ επιτρέπει στατιστική γενίκευση
- Τύποι: Ευκαιριακή, Ποσοστική, Σκόπιμη, Χιονοστιβάδας

### Μέγεθος Δείγματος vs Τύπος:
> **Κανόνας**: Μικρό πιθανοτικό δείγμα > Μεγάλο μη πιθανοτικό δείγμα

Γιατί; Μόνο με πιθανοτική δειγματοληψία μπορούμε να:
- Υπολογίσουμε το δειγματοληπτικό σφάλμα
- Γενικεύσουμε στον πληθυσμό
- Χρησιμοποιήσουμε στατιστικά τεστ

### Πιθανότητα Επιλογής σε Exit Poll:
Αν πληθυσμός = 10.000.000 ψηφοφόροι και δείγμα = 1.000:
- Πιθανότητα επιλογής = 1.000 / 10.000.000 = 0,0001 = 0,01%
- Πολύ χαμηλή πιθανότητα για οποιονδήποτε συγκεκριμένο!

## 7. Δειγματοληπτικά Πλαίσια
*Απαραίτητο για: Ερώτηση 8*

### Τι είναι Δειγματοληπτικό Πλαίσιο;
Η **λίστα** ή **καταγραφή** του πληθυσμού από την οποία επιλέγουμε το δείγμα.

### Προβλήματα Τηλεφωνικών Καταλόγων:

| Πρόβλημα | Επίπτωση |
|----------|----------|
| Εξαιρούν κινητά | Χάνονται νέοι, κινητικοί |
| Εξαιρούν μη καταχωρημένους | Υπο-αντιπροσώπευση |
| Μεροληψία κοινωνικής τάξης | Φτωχοί = λιγότερες γραμμές |
| Νέοι συνδρομητές | Δεν έχουν καταχωρηθεί |
| Παλαιότητα | Ξεπερασμένα στοιχεία |

### Ιδανικό Δειγματοληπτικό Πλαίσιο:
- Πλήρες (περιλαμβάνει όλον τον πληθυσμό)
- Ακριβές (σωστά στοιχεία)
- Ενημερωμένο (τρέχουσα κατάσταση)
- Χωρίς διπλοεγγραφές

## 9. Ηθικά Ζητήματα στην Ποιοτική Έρευνα
*Απαραίτητο για: Ερώτηση 10*

### Βασικά Ηθικά Ζητήματα:

#### 1. Υποκειμενικότητα στην Ανάλυση
- Κίνδυνος να βρούμε αυτό που θέλουμε
- Επίδραση προσωπικών αξιών/πεποιθήσεων

**Αντιμετώπιση:**
- Αυτογνωσία (reflexivity)
- Τήρηση καθιερωμένων τεχνικών
- Επισκόπηση από συναδέλφους (peer review)
- Τριγωνοποίηση δεδομένων

#### 2. Προστασία Ιδιωτικότητας
- Αναγνώριση συμμετεχόντων
- Ευαίσθητες πληροφορίες
- Μικρά δείγματα = εύκολη ταυτοποίηση

**Αντιμετώπιση:**
- Ανωνυμοποίηση
- Αφαίρεση αναγνωριστικών στοιχείων
- Συγκατάθεση (informed consent)

#### 3. Σχέση με Συμμετέχοντες
- Μακροχρόνια επαφή
- Εμπιστοσύνη vs αντικειμενικότητα
- Δύναμη του ερευνητή

## 10. Πίνακες Συνάφειας (Crosstabulation)
*Απαραίτητο για: Ερωτήσεις 11, 13*

### Βήματα Δημιουργίας:

1. **Καταμέτρηση**: Μετράμε τις περιπτώσεις για κάθε συνδυασμό
2. **Σύνολα**: Υπολογίζουμε σύνολα γραμμών και στηλών
3. **Ποσοστά**: Υπολογίζουμε % (συνήθως κατά στήλη)
4. **Ερμηνεία**: Συγκρίνουμε τα ποσοστά

### Δομή Πίνακα:

```
                    ΑΝΕΞΑΡΤΗΤΗ ΜΕΤΑΒΛΗΤΗ
                    Κατηγ.1   Κατηγ.2   Κατηγ.3   Σύνολο
ΕΞΑΡΤΗΜΕΝΗ  Κατηγ.Α    n(%)      n(%)      n(%)      n
ΜΕΤΑΒΛΗΤΗ   Κατηγ.Β    n(%)      n(%)      n(%)      n
            Σύνολο     n(100%)   n(100%)   n(100%)   N
```

### Οδηγίες Ερμηνείας:
- Συγκρίνουμε ποσοστά **μεταξύ στηλών** (κατηγορίες ανεξάρτητης)
- Αναζητούμε **μοτίβα** (ποια ομάδα έχει υψηλότερα/χαμηλότερα %)
- Σχολιάζουμε τις **διαφορές**

## 11. Κωδικοποίηση και Βιβλίο Κωδικών
*Απαραίτητο για: Ερώτηση 12*

### Τι είναι το Βιβλίο Κωδικών (Codebook);
Έγγραφο που περιγράφει:
- Τις **θέσεις** των μεταβλητών
- Τους **κωδικούς** για κάθε τιμή
- Τη διαχείριση **ελλιπών τιμών**

### Δομή Βιβλίου Κωδικών:

| Μεταβλητή | Κωδικός | Τιμή |
|-----------|---------|------|
| Φύλο | 1 | Άνδρας |
|  | 2 | Γυναίκα |
|  | 9 | Καμία απάντηση |
| Ηλικία | 01-98 | Πραγματική ηλικία |
|  | 99 | Καμία απάντηση |

### Κανόνες Κωδικοποίησης:
1. **Αριθμητικοί κωδικοί** (όχι γράμματα)
2. **Συνέπεια** σε όλο το αρχείο
3. **Ειδικοί κωδικοί** για ελλιπή (π.χ. 9, 99, 999)
4. **Αμοιβαία αποκλειστικοί** κωδικοί
