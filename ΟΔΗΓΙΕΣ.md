# Κουίζ Μεθοδολογίας Εκπαιδευτικής Έρευνας

## 📁 Περιεχόμενα

- `quiz-methodology-standalone.html` - **Standalone έκδοση** (ανοίγει απευθείας στον browser)
- `quiz-methodology.jsx` - React component για ενσωμάτωση σε React project

---

## 🚀 ΕΠΙΛΟΓΗ 1: Άμεση Εκτέλεση (Πιο Εύκολη!)

### Απλά άνοιξε το HTML αρχείο!

1. Κατέβασε το αρχείο `quiz-methodology-standalone.html`
2. Κάνε διπλό κλικ για να ανοίξει στον browser
3. Έτοιμο! 🎉

**Σημείωση:** Απαιτείται σύνδεση στο internet για τη φόρτωση του Tailwind CSS και του React.

---

## 💻 ΕΠΙΛΟΓΗ 2: Τοπική Εκτέλεση με React (Vite)

### Βήμα 1: Εγκατάσταση Node.js
Κατέβασε και εγκατέστησε το Node.js από: https://nodejs.org/

### Βήμα 2: Δημιουργία Project
```bash
# Δημιουργία νέου React project με Vite
npm create vite@latest quiz-methodology -- --template react

# Μετάβαση στον φάκελο
cd quiz-methodology

# Εγκατάσταση dependencies
npm install
```

### Βήμα 3: Προσθήκη Tailwind CSS
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Επεξεργασία `tailwind.config.js`:
```javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Επεξεργασία `src/index.css`:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### Βήμα 4: Αντικατάσταση Κώδικα
1. Αντέγραψε το περιεχόμενο του `quiz-methodology.jsx`
2. Επικόλλησέ το στο `src/App.jsx`
3. Πρόσθεσε στην αρχή: `import './index.css'`

### Βήμα 5: Εκτέλεση
```bash
npm run dev
```
Άνοιξε: http://localhost:5173

## 🌐 ΕΠΙΛΟΓΗ 3: Δημοσίευση Online (ΔΩΡΕΑΝ)

### A) GitHub Pages (Για το HTML αρχείο)

1. **Δημιουργία Repository στο GitHub**
   - Πήγαινε στο https://github.com/new
   - Όνομα: `quiz-methodology`
   - Δημόσιο repository

2. **Ανέβασμα αρχείου**
   - Κάνε κλικ "Add file" > "Upload files"
   - Μετονόμασε το HTML σε `index.html` πριν το ανεβάσεις
   - Commit changes

3. **Ενεργοποίηση GitHub Pages**
   - Settings > Pages
   - Source: "Deploy from a branch"
   - Branch: `main` / `root`
   - Save

4. **Το site θα είναι διαθέσιμο σε:**
   ```
   https://[username].github.io/quiz-methodology/
   ```

### B) Netlify (Πιο Εύκολο - Drag & Drop!)

1. Πήγαινε στο https://netlify.com
2. Κάνε εγγραφή (δωρεάν)
3. Πήγαινε στο "Sites"
4. **Σύρε και άφησε** το φάκελο με το HTML αρχείο
5. Έτοιμο! Θα πάρεις ένα link όπως:
   ```
   https://random-name-12345.netlify.app
   ```

### C) Vercel (Για React Projects)

1. Εγκατάσταση Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Στον φάκελο του project:
   ```bash
   vercel
   ```

3. Ακολούθησε τις οδηγίες
4. Θα πάρεις ένα link όπως:
   ```
   https://quiz-methodology.vercel.app
   ```

### D) Surge.sh (Απλό & Γρήγορο)

```bash
# Εγκατάσταση
npm install -g surge

# Δημοσίευση (στον φάκελο με το HTML)
surge

# Θα σου ζητήσει email και password
# Θα πάρεις link: https://random-name.surge.sh
```

## 🔧 Προσαρμογές

### Αλλαγή Χρωμάτων
Στο HTML αρχείο, άλλαξε τα χρώματα του Tailwind:
- `bg-blue-600` → `bg-green-600` (για πράσινο)
- `from-blue-50 to-indigo-100` → `from-green-50 to-teal-100`

### Προσθήκη Ερωτήσεων
Στο array `questions`, πρόσθεσε νέα αντικείμενα:
```javascript
{
  id: 48,
  question: "Η νέα σου ερώτηση εδώ",
  options: [
    "α. Επιλογή 1",
    "β. Επιλογή 2",
    "γ. Επιλογή 3",
    "δ. Επιλογή 4",
    "ε. Επιλογή 5"
  ],
  correct: 2  // 0-based index (γ = 2)
}
```

### Τυχαία Σειρά Ερωτήσεων
Πρόσθεσε πριν το `return`:
```javascript
const shuffledQuestions = [...questions].sort(() => Math.random() - 0.5);
```
