## 1. Αυτοαξιολόγηση με Flashcards

### Ιδέα:
Δημιουργία καρτών με την ερώτηση στη μία πλευρά και τα βασικά σημεία απάντησης στην άλλη.

### Εφαρμογές για Flashcards:
- **Anki** (δωρεάν) - Με spaced repetition
- **Quizlet** (δωρεάν με premium) - Online
- **Notion** (δωρεάν) - Με toggle blocks

### Παράδειγμα Κάρτας:
```
ΜΠΡΟΣΤΑ:
"Ποιοι είναι οι τρεις σκοποί διεξαγωγής έρευνας;"

ΠΙΣΩ:
1. Διερευνητική (ΤΙ υπάρχει)
2. Περιγραφική (ΠΟΣΟ, ΠΟΥ, ΠΟΤΕ)
3. Ερμηνευτική (ΓΙΑΤΙ)
```

## 2. Διαδραστικό Quiz με Αυτοβαθμολόγηση

### Τύπος A: Αντιστοίχιση Εννοιών

**Οδηγία:** Αντιστοιχίστε την έννοια με τον σωστό ορισμό

| # | Έννοια | | Ορισμός |
|---|--------|---|---------|
| 1 | Μεταβλητή | | A. Τι/ποιον μελετάμε |
| 2 | Χαρακτηριστικά | | B. Λίστα του πληθυσμού |
| 3 | Μονάδα ανάλυσης | | C. Χαρακτηριστικό με διαφορετικές τιμές |
| 4 | Δειγματοληπτικό πλαίσιο | | D. Τιμές μιας μεταβλητής |

**Απάντηση:** 1-C, 2-D, 3-A, 4-B

---

### Τύπος B: Συμπλήρωση Κενών

**Ερώτηση 1:**
Η __________ έρευνα απαντά στο ερώτημα "ΓΙΑΤΙ;" και αναζητά __________ σχέσεις.

**Απάντηση:** Ερμηνευτική, αιτιακές

**Ερώτηση 2:**
Στην εννοιολόγηση, πρώτα ορίζουμε τις __________, μετά επιλέγουμε τους __________ για κάθε μία.

**Απάντηση:** διαστάσεις, δείκτες

**Ερώτηση 3:**
Η κλίμακα μέτρησης με απόλυτο μηδέν λέγεται __________.

**Απάντηση:** αναλογική

---

### Τύπος Γ: Σωστό/Λάθος με Αιτιολόγηση

| Πρόταση | Σ/Λ | Αιτιολόγηση |
|---------|-----|-------------|
| Η ποσοστική δειγματοληψία είναι πιθανοτική | | |
| Η κλίμακα Likert είναι διατακτική | | |
| Οι τηλεφωνικές έρευνες δεν έχουν μεροληψία συνεντευκτή | | |
| Μεγαλύτερο δείγμα = πάντα καλύτερα | | |

**Απαντήσεις:**
1. Λ - Είναι μη πιθανοτική
2. Σ - Έχει σειρά αλλά όχι απαραίτητα ίσα διαστήματα
3. Λ - Υπάρχει (φωνή, τρόπος ανάγνωσης)
4. Λ - Η πιθανοτική δειγματοληψία είναι πιο σημαντική

### Άσκηση 2: Εννοιολόγηση

**Οδηγία:** Επιλέξτε μία από τις παρακάτω έννοιες και αναπτύξτε την εννοιολόγηση:
- Σχολική βία
- Ακαδημαϊκή επιτυχία
- Οικονομική ανασφάλεια

**Πρότυπο απάντησης:**
1. Ορισμός της έννοιας
2. 3-4 Διαστάσεις
3. 2-3 Δείκτες για κάθε διάσταση
4. Παράδειγμα ερώτησης για κάθε δείκτη

## 4. Τεχνική Pomodoro για Μελέτη

### Πρόγραμμα Μελέτης (5 ημέρες):

| Ημέρα | Θεματική | Χρόνος |
|-------|----------|--------|
| 1 | Μεταβλητές, Σκοποί, Εννοιολόγηση | 2 pomodoros |
| 2 | Κλίμακες μέτρησης, Ελλιπή δεδομένα | 2 pomodoros |
| 3 | Δειγματοληψία (όλα τα είδη) | 2 pomodoros |
| 4 | Μέθοδοι έρευνας, Ηθική | 2 pomodoros |
| 5 | Πίνακες, Κωδικοποίηση, Επανάληψη | 2 pomodoros |

*1 pomodoro = 25 λεπτά μελέτη + 5 λεπτά διάλειμμα*

## 5. Ομαδική Μελέτη

### Δραστηριότητα: "Καυτή Καρέκλα"

**Διαδικασία:**
1. Ένας παίρνει το ρόλο του "εξεταστή"
2. Οι υπόλοιποι κάθονται στην "καυτή καρέκλα" με τη σειρά
3. Ο εξεταστής διαβάζει μια ερώτηση
4. Ο εξεταζόμενος έχει 2 λεπτά να απαντήσει
5. Οι υπόλοιποι συμπληρώνουν

### Δραστηριότητα: "Διδάσκω για να Μάθω"

**Διαδικασία:**
1. Κάθε μέλος αναλαμβάνει 2-3 ερωτήσεις
2. Μελετά σε βάθος τη θεωρία
3. "Διδάσκει" τους άλλους
4. Απαντά ερωτήσεις της ομάδας

## 8. Πρότυπο Αυτοελέγχου

### Checklist πριν τις Εξετάσεις:

#### Βασικές Έννοιες
- [ ] Μπορώ να ορίσω μεταβλητή vs χαρακτηριστικά
- [ ] Ξέρω τους 3 σκοπούς έρευνας
- [ ] Καταλαβαίνω τι είναι εννοιολόγηση

#### Κλίμακες Μέτρησης
- [ ] Διακρίνω τα 4 επίπεδα
- [ ] Μπορώ να δώσω παράδειγμα για κάθε επίπεδο
- [ ] Ξέρω ποιες πράξεις επιτρέπονται σε κάθε επίπεδο

#### Δειγματοληψία
- [ ] Ξεχωρίζω πιθανοτική vs μη πιθανοτική
- [ ] Ξέρω πότε να χρησιμοποιήσω κάθε τύπο
- [ ] Καταλαβαίνω τι είναι δειγματοληπτικό πλαίσιο

#### Μέθοδοι Έρευνας
- [ ] Γνωρίζω πλεονεκτήματα/μειονεκτήματα κάθε μεθόδου
- [ ] Μπορώ να επιλέξω κατάλληλη μέθοδο για σενάριο

#### Ανάλυση Δεδομένων
- [ ] Μπορώ να φτιάξω πίνακα συνάφειας
- [ ] Ξέρω να ερμηνεύω ποσοστά
- [ ] Καταλαβαίνω τι είναι βιβλίο κωδικών
